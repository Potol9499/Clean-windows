
一款簡單到不行的windows緩存清理工具帶有GUI



該程式特點:
```
1.極其簡單你不需要安裝開箱即可使用
2.沒有連線網路不需要擔心資料洩漏
3.他有GUI讓您方便超做
4.帶有管理員權限檢測
```
